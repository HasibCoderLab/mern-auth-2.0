# MERN-Auth 2.0
![ MERN-Auth 2.0](https://img.shields.io/badge/version-2.0-blue.svg)



# node initialized
```
npm init -y
```
# Create Server For Express 
```
npm i express
```
# Server Auto Restart
```
npm i nodemon
```
# Environment Variables (Security features)

```
npm i dotenv
```
# Mongoose(MongoDB)
```
npm i mongoose
```

# Json Web token (jwt)
```
npm i jwt
```

# Cookie-Parser
```
npm i cookie-parser
```
